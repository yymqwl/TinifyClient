# TinifyClient

TinifyClient是一个调用tinify-net示例程序.使用TinifyClient可以调用tinify-net直接压缩图片文件。
关于TinyPng的链接https://tinypng.com/

1.安装Vs,和.NetCore
2.执行编译命令得到TinifyClient.exe控制台文件
3.TinifyClient.exe "key"  "InputPath"   "OutputPath"
4.参数1是需要在https://tinypng.com/developers申请key填入。参数2是填写图片文件夹路径  参数3填写输出文件夹路径
5.执行后需要等待执行完成后。图片数量限制详情参考https://tinypng.com/
